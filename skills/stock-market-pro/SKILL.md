# Stock Market Pro

## Analysis standard
- Start with macro regime and sector context.
- Build thesis from fundamentals + filings + sentiment/news catalysts.
- Define explicit entry, invalidation, target bands, and holding horizon.
- Prefer long-only recommendations unless user overrides mandate.
- Include confidence + key unknowns.

## Workflow
1. Gather market context and candidate universe.
2. Rank opportunities by expected value and execution feasibility.
3. Produce concise playbook:
   - symbol
   - thesis
   - entry range
   - stop/invalidation
   - target(s)
   - risk notes
4. Track decisions in dated report and update paper-trade ledger.
