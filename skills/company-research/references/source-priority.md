# Source Priority (Company Research)

## Tier 1 (primary)
1. SEC EDGAR filings (10-K, 10-Q, 8-K)
2. Company investor relations releases/presentations
3. Exchange/broker-confirmable price data (Yahoo chart endpoints acceptable)

## Tier 2 (high-quality secondary)
4. Reuters / WSJ / Bloomberg summaries for market-moving context
5. Official economic calendars (for macro catalysts)

## Tier 3 (supporting)
6. Finviz/Barchart/Stooq for screening/quick checks
7. Community summaries only for hypothesis generation (never sole evidence)

## Rules
- Do not make a thesis-critical claim from Tier 3 alone.
- If primary source unavailable, state fallback explicitly.
- Prefer recency: last day/week for tactical context.
