# Daily 1% Improvement Loop

After each report/trade cycle:

1. **Score quality (0-2 each):**
   - Thesis quality
   - Entry timing
   - Risk sizing/control
   - Execution clarity
2. **RCCA entry:**
   - Root cause
   - Corrective action
   - Rule/prompt update
3. **Promotion logic:**
   - If same correction repeats 3x in <=30 days, propose permanent rule.
4. **Pruning logic:**
   - If a rule is stale or unused for 30+ days, demote to archive candidate.

Always cite where a rule came from (file + line) when applying it.
