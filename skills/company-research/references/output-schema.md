# Required Output Schema (Actionable)

For each ticker, output in this order:

1. **Classification:** ENTER TODAY | HOLD | SELL | WATCHLIST
2. **Thesis (3 bullets max):** fundamental driver, catalyst, regime fit
3. **Evidence:** filing-derived facts + current market context (source-cited)
4. **Levels:**
   - Entry trigger/zone
   - Stop-loss
   - Take-profit target(s)
   - Invalidation condition
5. **Risk Box:** top 3 disconfirming risks
6. **Confidence:** 0-100 with one-line reason
7. **RCCA hook:** if prior call exists, what changed vs last thesis

Portfolio-level footer:
- Macro regime summary (rates/dollar/oil/vol + key catalyst)
- Sizing note
- Rule changes applied today (if any)
